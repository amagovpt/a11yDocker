# a11yDocker
The best way to have our tools in your local environment

Our tools running on your local server:

- [AccessMonitor](https://accessmonitor.acessibilidade.gov.pt) - a WCAG automatic evaluator - to create an accessibility report of one webpage 
- [MyMonitor](https://mymonitor.acessibilidade.gov.pt) - a WCAG automatic evalutor - to monitor various websites
- [Observatory](https://observatorio.acessibilidade.gov.pt) - a WCAG dashboard of all your websites - a showcase of the automatic monitoring of your websites
- Admin Monitor Suite - backoffice to manage MyMonitor and Observatory
